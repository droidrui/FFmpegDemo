# FFmpegDemo

### 这是一个使用FFmpeg编解码音视频的Demo。
### 包含以下示例：

* [x] 解码音频

* [ ] 解码视频

* [ ] 编码音频

* [ ] 编码视频

### 详细说明：

#### 解码音频

1. 边解码mp3音频边保存pcm文件边使用OpenSL播放

2. 使用pthread线程锁和条件变量同步解码和播放

3. 使用面向对象C封装FFmpeg解码器和OpenSL播放器

#### 解码视频

1. 边解码mp4视频边播放边保存yuv文件

#### 编码音频

1. 读取OpenSL录音数据实时编码保存为aac文件

#### 编码视频

1. 读取手机摄像头数据实时编码保存为flv文件
